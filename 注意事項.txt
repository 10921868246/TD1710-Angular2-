Angular2 程式範例說明：

1.本專案所有範例並無包含模組套件「node_modules」，
　請開啟命令提示字元(CMD)，切至該專案目錄底下，
　並執行 npm 命令：

============================
npm install
============================


2.執行完畢後直接透過 Angular-Cli 執行即可：

============================
ng serve
============================


3.若未安裝 Angular-Cli 者，請執行 npm 命令：

============================
npm install -g @angular/cli
============================
